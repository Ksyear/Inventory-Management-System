# 상품 재고 관리 시스템

## 상품 관리 정보(Product)
- 상품 ID
- 상품명
- 카테고리
- 가격
- 재고 수량
- 품절 여부

## 상품 목록 관리 기능(ShopManager)
- 상품 추가
- 상품 삭제
- 상품명 수정
- 카테고리 수정
- 상품 가격 수정
- 상품 재고 수정
- 품절 여부 수정(품절 여부 설정)
- 전체 상품 정보 출력
- 상품명으로 검색
- 카테고리로 검색
- 상품 가격으로 검색
- 상품 재고로 검색
- 품절로 검색
- 카테고리별 상품 정보 출력 // 이거 어케해야될까
- 가격 낮은 순으로 정렬해서 출력
- 재고가 n개 이하인 상품 정보 출력
- 카테고리별 상품 가격 평균 출력
